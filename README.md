## 🧑‍🎨 About Acornriver

**기억, 언어, 그리고 기술의 교차점.**
저는 텍스트와 미디어 아트를 통해 보이지 않는 감각과 기억을 시각화하는 작업을 합니다.
단순한 기술적 구현을 넘어, 인지심리학적 관점과 인문학적 상상력을 결합하여 관객에게 새로운 경험을 제시합니다.

* **Role:** Media Artist, Art Director, Writer
* **Focus:** Installation Art, Interactive Media, Creative Coding, Writing
* **Location:** South Korea
