# 프로젝트 추가 및 관리 매뉴얼

이 프로젝트는 `asset/WebIndependence` 폴더의 내용을 바탕으로 웹사이트 데이터를 자동으로 생성합니다.

## 1. 새 프로젝트 추가 방법

1. `asset/WebIndependence` 폴더 안에 **새로운 폴더**를 만듭니다. (예: `16새로운프로젝트`)
2. 폴더 안에 다음 파일들을 넣습니다:

### 필수 파일
- **info.json**: 프로젝트 정보를 담은 파일
  ```json
  {
    "id": "p16",
    "title": "16 새로운 프로젝트",
    "year": "2025",
    "category": "New Media Art"
  }
  ```
- **description.md**: 프로젝트 상세 설명 (마크다운 형식)
- **이미지 파일들**: `.jpg`, `.png` 등
  - **썸네일**: 파일명에 `썸네일` 또는 `thumbnail`이 포함된 이미지는 자동으로 썸네일로 지정됩니다.
  - **갤러리**: 나머지 이미지들은 자동으로 상세 페이지의 갤러리에 추가됩니다.

## 2. 웹사이트 업데이트 (빌드)

파일을 추가하거나 수정한 후에는 반드시 아래 명령어를 실행해야 웹사이트에 반영됩니다.

터미널에서 다음 명령어를 입력하세요:

```bash
npm run build
```
또는
```bash
node build.mjs
```

이 명령어를 실행하면 `data.js` 파일이 자동으로 갱신됩니다.

## 3. CV (이력서) 수정

`asset/WebIndependence/CV/cv.md` 파일을 수정하면 이력서 내용이 업데이트됩니다. 수정 후 마찬가지로 빌드 명령어를 실행해주세요.
